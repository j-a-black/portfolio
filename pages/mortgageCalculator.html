Purpose and Objectives

The purpose of the project was to create a simple app that would allow to me expand my understanding of State and Event Listeners. My objectives were to:

  • 1. Initialize state containing the preset values declared in the input fields
  • 2. Update the state using an event handler that listens for input
  • 3. Display the dynamic changes in values using a pie chart from Chart.js

Challenges

When using an event handler to update the state, my knowledge of the type of information events pass was very limited. I conducted searches to first learn what an event is, followed by the properties available in the event interface, and lastly, how using the target property to specifically target the name and value of the element can be used to update the state.

In time, I learned that event.target.name and event.target.value were the two event properties that contained the information needed to update the state.
